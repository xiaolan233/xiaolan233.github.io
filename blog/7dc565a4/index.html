单选

1单选

传递函数定义为（）

A.在线性定常系统中，当初始条件为零时，系统输出的拉氏变换与输入拉氏变换之比
B.$G(s)=\dfrac{C(s)}{R(s)}$
C.系统输出的拉氏变换与输入的拉氏变换之比
D.以上三者都是

正确答案：A你选对了

2单选

传递函数是经典控制理论的数学模型之一，它具有如下特点（ ）。

A.传递函数与系统微分方程式之间不可以相互转换
B.它可以反映出系统输入输出之间的动态性能，但不能反映系统结构参数对输出的影响
C.传递函数表示系统传递、变换输入信号的能力，不仅与系统的结构和参数有关，而且与输入输出信号形式有关
D.传递函数表示系统传递、变换输入信号的能力，只与系统的结构和参数有关，与输入输出信号形式无关

3单选

信号流图是自动控制系统的数学模型之一，根据信号流图求解输入和输出信号之间传递函数的方法是（ ）

A.以上都不对
B.已知信号流图，应用梅森公式可直接求得系统的传递函数
C.已知系统动态结构图，应用梅森公式不可直接求系统的传递函数
D.已知信号流图，应用梅森公式不可直接求系统的传递函数

正确答案：B你选对了

4单选

传递函数表示系统传递、变换输入信号的能力，与系统的结构和参数有关，与（ ）无关。

A.与输出信号的形式无关，与输入信号的形式有关
B.与输入和输出信号的形式无关
C.与输入和输出信号的形式有关
D.与输入信号的形式无关，与输出信号的形式有关

正确答案：B你选对了

5单选

对同一个系统，取不同的物理量作为系统的输入和输出时，传递函数（ ）

A.通常相同
B.通常不同
C.传递函数相同，但微分方程模型不同
D.传递函数不同，但微分方程模型相同

正确答案：B你错选为C

6单选

一阶微分环节微分方程为( )

A.$Ts+1$
B.$\dfrac{1}{Ts+1}$
C.$c(t)=\dfrac{dr(t)}{dt}$
D.$c(t)=T\dfrac{dr(t)}{dt}+r(t)$

正确答案：D你错选为C

7单选

振荡环节的传递函数为( )
A.$Ts+1$
B.$\dfrac{1}{Ts+1}$
C.$G(s)=\dfrac{\omega^{2}_n}{s^2+2\zeta\omega_ns+\omega^2_n}$
D.$\dfrac{1}{s}$
正确答案：C你错选为B

8单选

当多个环节串联连接时，其传递函数为多个环节传递函数的( )

A.以上都不对
B.乘积
C.代数和
D.相比较

正确答案：B你选对了

9单选

关于传递函数，下列说法正确的是

A.只有初始条件为零的线性定常系统才有传递函数
B.初始条件为零的系统就有传递函数
C.只要是线性定常系统就有传递函数
D.所有线性系统都有传递函数

正确答案：C你选对了

10单选

已知系统的原理图如图所示，其中$r(t)$和$c(t)$分别为对应点的位移，$k1$和$k2$分别为对应弹簧的弹性系数，$f$为阻尼器的阻尼系数。以$r(t)$为输入、$c(t)$为输出，系统的传递函数为

A.$\dfrac{fk_1s}{f(k_1+k_2)s+k_1k_2}$
B.$\dfrac{fk_2s}{f(k_1+k_2)s+k_1k_2}$
C.$\dfrac{fk_1s}{f(k_1+k_2)s+k_1+k_2}$
D.$\dfrac{fk_1s}{fk_1k_2s+k_1k_2}$

正确答案：A你错选为C

多选

11多选

自动控制系统的数学模型包括（ ）

A.微分方程

B.信号流图

C.热学方程

D.传递函数
正确答案：A、B、D你错选为A、D

12多选

自动控制系统的典型环节有（ ）

A.一阶微分环节

B.比例环节

C.二阶微分环节

D.正弦环节
正确答案：A、B、C你选对了

13多选

自动控制系统的动态结构图表示了系统中信号的传递和变换关系，经过等效变换可以求出系统输入与输出之间的关系，对于复杂系统，还有相互交错的局部反馈，必须经过相应的变换才能求出系统的传递函数，等效变换方法有（ ）

A.引出点合并、综合点的合并

B.综合点的前移和后移

C.引出点的前移和后移

D.串联连接、并联连接、反馈连接的变换
正确答案：A、B、C、D你选对了

14多选

在建立系统微分方程模型时，( )

A.微分方程的标准型要求：将与输出量有关的各项放在等号的右边，而与输入量有关的各项放在等号的左边，各导数项按降价排列

B.不须确定系统的输入量和输出量

C.必须确定系统的输入量和输出量

D.微分方程的标准型要求：将与输入量有关的各项放在等号的右边，而与输出量有关的各项放在等号的左边，各导数项按降价排列
正确答案：C、D你错选为A、C

15多选

传递函数是经典控制理论的数学模型之一，它具有如下特点( )

A.传递函数与系统微分方程之间可以相互转换

B.传递函数表示系统传递、变换输入信号的能力，只与系统的结构和参数有关，与输入输出信号的形式无关

C.传递函数表示系统传递、变换输入信号的能力，不仅与系统的结构和参数有关，而且与输入输出信号的形式有关

D.它可以反映出系统输入输出之间的动态性能，也可以反映出系统结构参数对输出的影响
正确答案：A、B、D你错选为A、B

16多选

下面有关信号流图的术语中，正确的是( )

A.只有输出支路的节点称为输入节点，只有输入支路的节点称为输出节点，既有输入支路又有输出支路的节点称为混合节点

B.节点表示系统中的变量或信号

C.支路是连接两个节点的有向线段，支路上的箭头表示信号传递的方向，传递函数标在支路上

D.前向通道为从输入节点开始到输出节点终止，且每个节点通过（一次）的通道，前向通道增益等于前向通道中各个支路增益的乘积
正确答案：A、B、C、D你错选为D

17多选

输入量与输出量成正比的环节称为( )

A.比例环节

B.$\dfrac{C(t)}{r(t)}=K$

C.微分环节

D.积分环节
正确答案：A、B你没选择任何选项

18多选

惯性环节又称为非周期环节，该环节中具有一个储能元件，惯性环节的数学模型为( )

A.$Ts+1$

B.$\dfrac{1}{Ts+1}$

C.$T\dfrac{dc(t)}{dt}+c(t)=r(t)$

D.$c(t)=\int r(t)dt$
正确答案：B、C你错选为C

19多选

理想微分环节的输出量与输入量的变化率成正比，其数学模型为( )

A.$s$

B.$c(t)=\dfrac{dr(t)}{dt}$

C.$c(t)=\int r(t)dt$

D.$\dfrac{1}{Ts+1}$
正确答案：A、B你错选为C

20多选

系统的开环传递函数为:

A.前向通道的传递函数除以1加上前向通道的传递函数与反馈通道的传递函数的乘积

B.开环系统的传递函数

C.闭环系统在反馈通路断开时输入与反馈信号间的传递函数

D.前向通道的传递函数与反馈通道的传递函数的乘积
正确答案：C、D你错选为D
