Hexo部署问题 Hexo -d时 `fatal bad config line 1 in file .gitconfig`

发表于 2018-08-31 | 更新于 2018-11-05 | 分类于 Hexo搭建博客 ， 遇到问题 | 评论数： | 阅读次数：

Hexo -d发布问题：fatal: bad config line 1 in file .git/config

应该是.git/config这个文件有问题。这个文件在哪里呢，就在.deploy_git目录下。所以：
解决方法:

  • 删除hexo站点目录下的.deploy_git目录，然后再输入hexo clean &&hexo g&&hexo d即可发布。

参考:https://blog.csdn.net/cijiancao/article/details/80819975
